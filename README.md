
# Calculadora de Interés Simple

Este proyecto es una calculadora básica de interés simple desarrollada en Python.

## 📌 Descripción

La calculadora permite ingresar un capital inicial, una tasa de interés y un período de tiempo para calcular el interés generado.

## 🧮 Fórmula utilizada

Interés Simple = Capital × Tasa × Tiempo

## 🚀 Ejecución

Ejecuta el archivo calculadora.py en Python.

## 💻 Ejemplo

Capital: 1000  
Tasa: 0.05  
Tiempo: 2  

Resultado: 100
